INSTRUCCIONES PARA CAMBIAR LAS IMÁGENES

1. Esta carpeta contiene las imágenes utilizadas en la aplicación.
2. Para cambiar una imagen, simplemente reemplazá el archivo existente con tu propia imagen, manteniendo el mismo nombre.

Archivos:
- hero.jpg: Imagen principal de la cabecera (sección "Elevá tu juego").
- about.jpg: Imagen de la sección "Conocé al Profe".

No es necesario editar ningún código si mantenés estos nombres de archivo.
